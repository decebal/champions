# Champions
Determine minimum no of matches to be removed for each team in order to become
champions

## Install

Via Git

``` bash
$ git clone https://github.com/solid-decebal/champions.git
$ composer install
```

## Usage

``` 
   php bin/footballer
```

## Requires
- MongoDb
